# Tic Tac Toe Web Game 

An interactive Tic Tac Toe game built using **HTML, CSS, and JavaScript** with multiple game modes, animated UI, and visual win detection.

---

##  Features

-  **Two Game Modes**
  - Player vs Player
  - Player vs Computer (AI)
-  **Attractive & modern UI**
-  **Smooth animations on every move**
-  **Winning strike line** across the board
-  **Clear win / draw messages**
-  Restart game anytime

---

##  Game Logic Overview

- Player **X** always starts first
- In AI mode:
  - Player = X
  - Computer = O (random valid moves)
- Game automatically detects:
  - Horizontal, vertical, diagonal wins
  - Draw conditions
- Winning pattern is highlighted using a **strike line**

---

##  Technologies Used

- **HTML** – Game structure
- **CSS** – Styling, animations, strike line
- **JavaScript** – Game logic, AI, DOM manipulation
